API-dokumentasjon

Bygg egendefinerte integrasjoner med Roasters kraftige REST API. Full dokumentasjon, webhooks og utviklerverktøy inkludert.

Kom i gang

# Autentiser med API-nøkkel
curl -X GET https://api.roaster.no/v1/schedules \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
